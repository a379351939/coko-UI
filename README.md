## 介绍

学习vue过程中做个的一个 UI 框架。

## 开始使用

1. 安装

使用框架前，在CSS中开启 border-box 
    ```
    *,*::before,*::after{box-sizing: border-box;}
    ```

2 .安装 gulu
    ```
    npm i --save tzc-2018920
    ```   
3 .引入 gulu
    ```
    import {Button, Group, Icon} from  'tzc-2018920'
    
    export default {
      name: 'app',
      components: {
        HelloWorld,
        'g-button': Button,
        'g-icon': Icon,
        'g-group': Group
      }
    }
    ```
